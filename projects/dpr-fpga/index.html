Dynamic Partial Reconfiguration on FPGA

Run-time adaptive reconfiguration of a RISC-V co-processor using Xilinx Vivado DFX.

Xilinx Vivado DFX RISC-V Partial Reconfiguration FPGA

Implemented run-time adaptive reconfiguration of a RISC-V processor's co-processor for resource optimisation on FPGA, using Xilinx Vivado Design Flow Extension (DFX). Dynamic partial reconfiguration allows specific FPGA regions to be reprogrammed at runtime without interrupting the rest of the design, enabling resource-efficient deployment of adaptive hardware.

Key aspects: - Designed reconfigurable partitions for the RISC-V co-processor - Implemented DFX flow in Xilinx Vivado with bitstream management - Resource optimisation through adaptive reconfiguration at runtime
