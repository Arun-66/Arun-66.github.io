This is the first post on this site. I'm going to use this space for three things:

1. **Reading notes.** Papers I've read carefully enough to summarise. Mostly mixed-signal design, RF circuits, and SoC architecture — but occasionally adjacent fields when something pulls me in.
2. **Half-formed thoughts.** Questions I'm working on that aren't ready to be papers, but are worth writing down so future-me can argue with present-me.
3. **Project write-ups.** Longer-form explanations of things I've built. The README on a repo only gets you so far.

No newsletter, no comments section, no rush. If something here is useful or wrong, my email is on the [about page](/).
