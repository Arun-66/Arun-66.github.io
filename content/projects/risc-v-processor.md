## Overview

A single-cycle implementation of the **RISC-V RV32I** base integer instruction set in Verilog. The processor was built as course material for the Computer Organization and Architecture course at PESU, where I served as a Teaching Assistant.

## Architecture

The datapath implements the five canonical stages as combinational logic in a single clock cycle:

1. **Instruction Fetch** — PC increments by 4 each cycle; branch/jump target computed separately
2. **Instruction Decode** — 7 primary opcode groups decoded; register file reads happen here
3. **Execute** — ALU performs arithmetic, logic, shift, and comparison operations; branch condition resolved
4. **Memory Access** — 32-bit word-addressable data memory with byte/halfword/word load-store
5. **Write Back** — MUX selects between ALU result, memory data, and PC+4 for register write

Control signals are generated by a combinational control unit with no pipeline registers.

## Validation

- Simulated using **RIPES** (a visual RISC-V simulator) for step-by-step instruction tracing
- Synthesised and tested on Xilinx FPGA using **Vivado**
- Test suite covers all 40 RV32I instructions including branch edge cases and load-store alignment

## Teaching Use

Used as the reference implementation for lab assignments where students extend the single-cycle design to a pipelined version with hazard detection.
