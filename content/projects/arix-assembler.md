A Python-based assembler for a custom ISA that takes assembly source code and produces hex output directly loadable into an instruction memory module in simulation or on FPGA.

Key aspects:
- Parses custom assembly mnemonics and operands
- Encodes instructions into binary and outputs hex format
- Supports labels, branches, and immediate values
- Designed as a companion tool for custom processor projects
