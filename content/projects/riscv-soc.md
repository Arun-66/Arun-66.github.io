Designed a RISC-V (RV32IM) core in Verilog, covering the full integer and multiply/divide ISA. The project includes FPGA validation and SoC integration with UART and SPI peripherals, enabling real hardware testing beyond simulation.

Key aspects:
- Full RV32IM instruction set implementation in Verilog
- FPGA validation on Xilinx boards using Vivado
- SoC integration with UART and SPI peripheral interfaces
- Ongoing development with planned expansion to additional peripherals
